| **太贵了** | 花生壳18元/台/年，1000台设备一年18000元 | ✅ **60元/月支持1000台设备**，比花生壳便宜25倍，比frp自建还便宜3倍 |
| **不好用** | frp只能单台配置，没有批量管理，没有监控 | ✅ **原生批量管理**，一键生成1000个设备接入签名许可证，直接烧录出厂 |

**这不是又一个frp克隆，是专门给中小OEM厂家做的批量远程运维基础设施。**

## ✨ MVP上线必带功能（没有任何多余功能）
- 任意TCP端口映射（支持SSH/HTTP/RTSP/Modbus等所有标准协议）
- 设备在线状态实时监控，离线自动重连
- 流量实时统计与历史账单查询
- 子账号权限管理与操作日志审计
- 批量证书管理（一键生成，支持工厂批量烧录）
- 完整RESTful API，可对接自有系统

**支持架构：** ARM32(armv5/armv6/armv7)、ARM64(aarch64)、MIPS(mips32/mips64)、x86(i386/amd64)  
（其他架构请联系告知，我们会优先评估适配）

## 🔒 安全说明
- 全程采用TLS 1.3加密传输，使用轻量级wolfssl库
- 每个用户独立证书，设备与用户账号一一绑定
- 设备端sheli程序采用MIT许可证开源，无后门，可自行编译审计
- 服务器间采用内网通信，核心服务不直接暴露公网
- 支持IP白名单，限制后台管理访问地址

*注：用户需妥善保管账号密码和设备证书，因保管不当导致的安全问题由用户自行承担*
