## 🎯 我们解决什么问题

设备卖出去后，一个小bug就要飞几千公里上门，一次成本2000+。
- 用frp自建：无法批量管理上千台设备，无监控无告警
- 用花生壳：1000台设备一年18000元，贵到离谱
- 用阿里云IoT：功能冗余复杂，计费不透明
- 设备是定制Linux/RTOS：主流工具根本无法适配

**ShellLink：设备出厂内置100KB小程序，通电即连，远程解决90%售后问题。**

## ✨ 上线必带功能

- 任意TCP端口映射（支持SSH/HTTP/RTSP/Modbus所有协议）
- 设备在线状态实时监控，离线自动重连
- 流量统计+历史查询
- 子账号权限管理+操作日志
- 批量证书管理（一键生成上千个，直接烧录）
- 远程固件升级（差分+断点续传+失败回滚）
- 设备异常告警（微信/企业微信推送）
- 完整RESTful API

**支持架构：ARM32/ARM64/MIPS/x86**
